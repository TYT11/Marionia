### Marionia 
------------

![](https://i.imgur.com/2MEDsSn.jpg)

網址：https://marionia.herokuapp.com/

#### Setup
------------
- Install packages: 
`npm install`

- Start the development server:
`npm start`

- Page runs at:
http://localhost:3000/

#### 介紹

------------

SCSS 手刻 RWD，實做照片輪播  
前、後端分離，部屬在 Heroku  
使用 React, Django Rest Framework
